Technologies Used

  • Angular - A powerful framework for building web applications with TypeScript.
  • TypeScript - A superset of JavaScript that adds static typing for better code organization and maintainability.
  • NgRx SignalStore - A robust state management solution for Angular, built with native Signal support. It offers a simple, declarative way to define stores, making state management more efficient and flexible.
  • Playwright - An end-to-end testing framework for web applications, enabling reliable and cross-browser testing.
  • Nx - A powerful monorepo management tool for building and managing large-scale projects with multiple applications and libraries.
  • Nx Cloud - A cloud-based service that accelerates builds and tests for Nx projects, improving developer productivity.
  • GitHub Actions - A CI/CD platform for automating builds, tests, and deployments of your projects.
  • Storybook - A development environment for building and documenting UI components, allowing for isolated component development and testing.
  • Storybook Interactions Tests - A feature of Storybook that enables testing component interactions and user flows.
  • Tailwind CSS - A utility-first CSS framework that makes it easy to build custom designs without writing a lot of CSS.
